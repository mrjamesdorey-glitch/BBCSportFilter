BBC Sport Filter - Changelog

Version 1.0 (October 17, 2025)

  • Initial release of BBC Sport Filter.
  • Added ability to filter BBC Sport content by category, team, or competition. .
  • Implemented local storage for filter persistence using chrome.storage.local.
  • Added content script to hide matching sports stories in real-time.
  • Included popup UI for adding, editing, and deleting filters.
